# prueba_tecnica_ADA_backend
Repositorio que contiene la prueba del backend propuesta por el equipo de ADA

Se tiene la carpeta con el repositorio para el CRUD de company, se adjunta en la carpeta de diversos, el modelo de la base de datos, el script de generación de las tablas, el script del procedimiento, para el llamado del cursor y la inserción en las tablas. Se adjunta archivo de calculadora.
